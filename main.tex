\textoresumo[english]{
\textbf{Context:} Modern software systems continuously grow in size, heterogeneity, and complexity, making program comprehension, maintenance, and quality assessment increasingly demanding. Although traditional static analysis, software visualization, and machine learning techniques have advanced considerably, many approaches still depend on language-specific parsers, expert-defined rules, or complex textual and structural representations. \textbf{Motivation:} This doctoral project investigates source code minimaps as compact visual signatures of software artifacts. The representation transforms source files into anonymized images that preserve structural, spatial, and syntactic cues, enabling the use of texture descriptors, convolutional neural networks, transfer learning, and explainable artificial intelligence. \textbf{Hypothesis:} Anonymized source code minimaps preserve enough visual and structural information to support scalable and language-agnostic software analysis while reducing dependence on manual rules and ecosystem-specific representations. \textbf{Objective:} The objective of this doctoral project is to define, evaluate, and operationalize a minimap-based approach for visual source code analysis, including dataset generation, artifact classification, model interpretation, and tool support inside development environments. \textbf{Preliminary results:} Previous studies indicate the feasibility of the approach with traditional classifiers based on Local Binary Patterns and with deep models trained on a corpus of 685,906 minimaps extracted from 100 open-source projects across 10 programming languages. The project also produced the MinimapSignatures ecosystem, composed of Visual Studio Code and IntelliJ IDEA plugins connected to a Python backend for inference. \textbf{Expected results:} The expected outcome is a reproducible methodology, datasets, and tools for analyzing source code through anonymized visual signatures, as well as further investigation of tasks related to software quality, architectural deviations, and code-review support.
}{Source code minimaps, Software visualization, Machine learning, Software analysis, Explainable artificial intelligence}
